# 🛒 Superstore Sales Analysis Dashboard

> **Internship Task | SkillCraft Technology**  
> Role: Data Analyst Intern  
> Tool Used: Microsoft Excel

---

## 📌 Project Overview

This project involves cleaning, analyzing, and visualizing a Superstore sales dataset as part of an internship assignment at **SkillCraft Technology**. The goal was to extract meaningful insights from raw data and present them through an interactive Excel dashboard.

---

## 🎯 Objectives

The task required fulfilling three key visualization conditions:

1. **Total Sales by Category** – Understand which product categories drive the most revenue
2. **Total Profit by Category** – Identify profitable vs. loss-making categories
3. **Sales Trend Over Time** – Analyze how sales have changed over a period of time

---

## 🧹 Data Cleaning

The raw Superstore dataset was cleaned using Excel, which included:

- Removing duplicate records
- Handling missing/null values
- Standardizing date formats
- Correcting inconsistent category/region labels
- Ensuring correct data types for numerical columns

---

## 📊 Dashboard Features

### Pivot Tables & Charts
- **Sales by Category** – Bar/Column chart showing total sales across product categories
- **Profit by Category** – Chart highlighting profit performance per category
- **Sales Trend Over Time** – Line chart displaying monthly/yearly sales progression

### Conditional Formatting
- Applied to the **Profit** column to visually distinguish:
  - 🟢 **Positive profit** – highlighted in green
  - 🔴 **Negative profit (loss)** – highlighted in red

### Interactive Dashboard
- All charts and pivot tables consolidated into a single, responsive Excel dashboard sheet
- Clean layout for easy interpretation and presentation
